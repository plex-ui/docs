### Auto resize

<ComponentPreview details={<>Enable <code>autoResize</code> to automatically grow/shrink the textarea as the user types. Control the bounds with <code>rows</code> (initial height) and <code>maxRows</code>.</>} code={`const [value, setValue] = useState("Line 1\\nLine 2\\nLine 3")
<div className="w-[400px]">
  <Textarea value={value} onChange={(e) => setValue(e.target.value)} placeholder="Type to grow..." autoResize rows={3} maxRows={8} />
</div>`} previewMinHeight={168}>
  <TextareaAutoResizeDemo />
</ComponentPreview>

### Sizing

<ComponentPreview details={<>Sizing of the textarea is controlled with <code>size</code> and <code>gutterSize</code> props. <code>size</code> sets the textarea to a specific control <code>height</code> for single-line equivalent, and <code>gutterSize</code> sets the horizontal padding.<br /><br />Each <code>size</code> provides default values for <code>gutterSize</code>, as well as <code>font-size</code> and <code>border-radius</code>. Each of these values can be overridden independently.</>} code={`<div className="flex flex-col gap-2.5 justify-start items-start w-[400px]">
  <Textarea placeholder="Write a message..." size="lg" />
</div>`} previewMinHeight={168}>
  <TextareaSizingDemo />
</ComponentPreview>

### Disabled

<ComponentPreview details={<>Use the <code>disabled</code> prop to disable the textarea, greying it out and preventing user interactions.</>} code={`<div className="w-[400px]">

  <TextareaDisabledDemo />

### Invalid

<ComponentPreview details={<>Use the <code>invalid</code> prop to mark the textarea as invalid. You decide how this state is removed, whether that's through <code>onFocus</code>, <code>onChange</code>, an <code>onSubmit</code> handler, etc.<br /><br />Default visual styles will be applied, based on the <code>variant</code>. Additionally, <code>[data-invalid]</code> is applied to the container for custom styling.</>} code={`<div className="w-[400px]">
  <Textarea invalid placeholder="Invalid textarea" />
</div>`} previewMinHeight={168}>
  <TextareaInvalidDemo />
</ComponentPreview>

### Auto select text

<ComponentPreview details={<>Pass <code>autoSelect</code> to select all text in the textarea &ndash; useful when pairing with other copy actions. This prop behaves like an imperative action when toggled and allows selection to change.</>} code={`<div className="w-[400px]">
  <Textarea defaultValue="Toggle to auto select" autoSelect={false} />
</div>`} previewMinHeight={168}>
  <TextareaAutoSelectDemo />
</ComponentPreview>

### Autofill extensions

<ComponentPreview details={<>By default, autofill extensions are prevented by <code>Textarea</code>. For the occasions where you want browser extensions to be present, pass <code>allowAutofillExtensions</code> to remove the disabling.<br /><br />When <code>name</code> is passed this value will default to <code>true</code>.</>} code={`<div className="w-[400px]">
  <Textarea key={String(allowAutofillExtensions)} name="email" placeholder={allowAutofillExtensions ? "Allowed" : "Not allowed"} allowAutofillExtensions={allowAutofillExtensions} />
</div>`} previewMinHeight={168}>
  <TextareaAutofillExtensionsDemo />
</ComponentPreview>
