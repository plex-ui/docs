### Sizing & roundness

<ComponentPreview details={<>
  <p>Control height with <code>size</code> and horizontal padding with <code>gutterSize</code>. Use <code>pill</code> for fully rounded corners.</p>
  <p><code>size</code> sets the button to a specific control height, and <code>gutterSize</code> sets the horizontal padding. Each <code>size</code> provides default values for <code>gutterSize</code> and <code>iconSize</code> props, as well as <code>font-size</code>. Each of these values can be overridden independently.</p>
  <p>When <code>pill</code> is set, extra horizontal gutter is dynamically applied to the button.</p>
</>} code={`<div className="flex flex-col gap-2 justify-start items-start">
  <Button color="primary" size="xl" pill><Mail /> Button <ArrowRight /></Button>
  <Button color="primary" size="xl" pill>Button <ArrowRight /></Button>
  <Button color="primary" size="xl" pill><Mail /> Button</Button>
  <Button color="primary" size="xl" pill>Button</Button>
</div>`} previewMinHeight={296}>
  <ButtonSizingDemo />
</ComponentPreview>

### Icons

<ComponentPreview details={<>
  <p>Add icons as children — sizing is automatic via <code>iconSize</code> prop.</p>
  <p>Default values for <code>iconSize</code> are provided through the <code>size</code> prop, but you can pass the prop directly to change the defaults.</p>
  <p>For icon-only buttons, there are two common approaches:</p>
  <p><strong>Square</strong> — create perfect square icon buttons with the <code>uniform</code> prop. When <code>uniform</code> is passed, <code>gutterSize</code> is ignored. Combine with <code>pill</code> to make circular icons.</p>
  <p><strong>Tighter gutters</strong> — applying a smaller <code>gutterSize</code> than <code>size</code> can create well-sized, but not exactly square, icon buttons.</p>
</>} code={`<Button color="secondary" size="lg" uniform variant="ghost">
  <PlusLg />
</Button>`} previewMinHeight={216}>
  <ButtonIconDemo />
</ComponentPreview>

### Inert

<ComponentPreview details={<>
  <p>Use <code>inert</code> to accessibly disable a button <strong>without any visual changes</strong>.</p>
  <p><strong>Note:</strong> For accessibility reasons, <code>disabled</code> is used instead of applying <code>inert</code> directly to the button. Visual disablement in the <code>disabled</code> prop is applied via <code>[data-disable]</code>.</p>

### Loading

<ComponentPreview details={<>
  <p>Use <code>loading</code> to display a loading indicator in the button, fading out the current content.</p>
  <p>Interactions will be prevented by default during this state, by providing a default value for <code>inert</code> to <code>true</code>. You can override this behavior directly with <code>inert</code>, or use <code>disabled</code> to add visual disablement.</p>
</>} code={`<Button loading size="xl" pill><ArrowUp /> Click to load</Button>`} previewMinHeight={168}>
