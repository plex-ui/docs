For form fields, use the [`Field`](/docs/components/field) component — it includes built-in label, description, and error handling.

## Usage

```tsx
import { Label } from "@plexui/ui/components/Label"
```

```tsx
<Label htmlFor="email">Your email address</Label>
```

## Label in Field

For stacked form rows with descriptions, errors, and horizontal layouts, use the [`Field`](/docs/components/field) component — it renders the label internally, hooks up `htmlFor` automatically, and handles the surrounding spacing.

## API Reference

See the [Radix UI Label documentation](https://www.radix-ui.com/primitives/docs/components/label) for the full prop surface.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `htmlFor` | `string` | — | `id` of the associated control. Clicks on the label focus or activate the control. |
| `className` | `string` | — | Applied to the root `<label>` element |

All other `<label>` attributes are forwarded.
