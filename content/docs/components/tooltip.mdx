### Base

### Different triggers

<ComponentPreview details={<><code>Tooltip</code> provides an optional component, <code>Tooltip.TriggerDecorator</code>, to help create accessible tooltip triggers. However, any component can be used as a trigger.</>} code={`<div className="flex items-center gap-4">
  <Tooltip content="Tooltip on button">
    <Button color="primary" size="lg">Sample button</Button>
  </Tooltip>
  <Tooltip content="Tooltip on span">
    <span>Inaccessible trigger</span>
  </Tooltip>
</div>`} previewMinHeight={168}>
  <div className="flex items-center gap-4">
    <Tooltip content="Tooltip on button">
      <Button color="primary" size="lg">Sample button</Button>
    </Tooltip>
    <Tooltip content="Tooltip on span">
      <span>Inaccessible trigger</span>
    </Tooltip>
  </div>
</ComponentPreview>

### Conditionally enabled

<ComponentPreview details={<>Sometimes a tooltip should only be disabled under certain conditions, such as when a component is disabled, providing more context as to why that is. Below is an example of how to do this by conditionally passing in <code>null</code> content.</>} code={`<Tooltip
  content={disabled ? "This action is disabled for reasons" : null}
>
  <Button color="primary" size="lg" disabled={disabled}>
    Sample button
  </Button>
</Tooltip>
`} previewMinHeight={168}>
  <TooltipConditionalDemo />
</ComponentPreview>

### Compact tooltips

<ComponentPreview details={<>Compact tooltips are designed for short, 1-3 word tooltips with inverted styling. They feature minimal padding, a dark background with white text, and no box shadows. Use <code>{'compact={true}'}</code> for brief status messages.</>} code={`<Tooltip content="Copy to clipboard" compact>
  <Button color="secondary" variant="ghost" size="md"><Copy /></Button>
</Tooltip>`} previewMinHeight={168}>
  <Tooltip content="Copy to clipboard" compact>
    <Button color="secondary" variant="ghost" size="md"><Copy /></Button>
  </Tooltip>
</ComponentPreview>
