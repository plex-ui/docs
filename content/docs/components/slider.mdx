## Examples

### Overview

<ComponentPreview code={`const [value, setValue] = useState(1000)
<Slider
  label="Example field"
  min={0}
  max={2000}
  resetValue={1000}
  step={10}
  unit="ms"
  value={value}
  onChange={setValue}
/>`} previewMinHeight={168}>
  <SliderBaseDemoWithControls />
</ComponentPreview>

### Adding a tooltip

<ComponentPreview details={<>A tooltip can easily be added to the Slider label by providing JSX instead of a simple string.</>} code={`<Slider
  label={
    <Tooltip content="More details about this field">
      <Tooltip.TriggerDecorator>Example field</Tooltip.TriggerDecorator>
    </Tooltip>
  }
  value={value}
  onChange={setValue}
  ...
/>`} previewMinHeight={168}>
  <SliderTooltipsDemo />
</ComponentPreview>

### Marks

<ComponentPreview details={<>Slider marks can be easily added by passing an array of values to the <code>marks</code> prop. Responsive behaviors are handled automatically.</>} code={`<Slider
  value={value}
  onChange={setValue}
  label="Passing grade"
  min={1}
  max={5}
  step={1}
  marks={[
    { value: 1, label: "1" },
    { value: 2, label: "2" },
    ...
  ]}
  trackColor="#61C454"
  rangeColor="#EF4146"
/>`} previewMinHeight={168}>
  <SliderMarksDemo />
</ComponentPreview>

### Range

<ComponentPreview details={<>Set <code>range</code> to enable a dual-thumb slider for selecting a value range. The <code>value</code> and <code>onChange</code> props accept arrays instead of single numbers.</>} code={`const [value, setValue] = useState([25, 75])
<Slider
  range
  label="Price range"
  min={0}
  max={100}
  step={1}
  value={value}
  onChange={setValue}
  prefixUnit="$"
/>`} previewMinHeight={168}>
  <SliderRangeDemo />
</ComponentPreview>

### Multiple thumbs

<ComponentPreview details={<>Pass three or more values to create a multi-thumb slider. Use <code>minStepsBetweenThumbs</code> to enforce minimum spacing between thumbs.</>} code={`const [value, setValue] = useState([10, 40, 70])
<Slider
  range
  label="Breakpoints"
  min={0}
  max={100}
  step={1}
  value={value}
  onChange={setValue}
  minStepsBetweenThumbs={5}
/>`} previewMinHeight={168}>
  <SliderMultipleThumbsDemo />
</ComponentPreview>

### Vertical

<ComponentPreview details={<>Set <code>orientation="vertical"</code> to render a vertical slider. Marks are not supported in vertical orientation.</>} code={`<Slider
  label="Volume"
  min={0}
  max={100}
  step={1}
  value={value}
  onChange={setValue}
  orientation="vertical"
/>`} previewMinHeight={280}>
  <SliderVerticalDemo />
</ComponentPreview>

### Disabled

<ComponentPreview details={<>Set <code>disabled</code> to prevent interaction with the slider.</>} code={`<Slider
  label="Disabled"
  min={0}
  max={100}
  step={1}
  value={50}
  onChange={() => {}}
  disabled
/>`} previewMinHeight={168}>
  <SliderDisabledDemo />
</ComponentPreview>
