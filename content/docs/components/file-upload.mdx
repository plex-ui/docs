## File Card

Standalone file card component. Wrap with `FileCardGroup` for the bordered container. Omit `onRemove` to hide the delete button.

### Read-only

<ComponentPreview code={`<FileCardGroup>
  <FileCard file={file} />
</FileCardGroup>`}>
  <FileCardDemo />
</ComponentPreview>

### With remove

<ComponentPreview code={`<FileCardGroup>
  <FileCard file={file} onRemove={() => removeFile()} />
</FileCardGroup>`}>
  <FileCardWithRemoveDemo />
</ComponentPreview>

### Multiple files

<ComponentPreview code={`<FileCardGroup>
  {files.map((file, i) => (
    <FileCard key={file.name} file={file} onRemove={() => removeFile(i)} />
  ))}
</FileCardGroup>`}>
  <FileCardListDemo />
</ComponentPreview>

### Read-only list

<ComponentPreview code={`<FileCardGroup>
  {files.map((file) => (
    <FileCard key={file.name} file={file} />
  ))}
</FileCardGroup>`}>
  <FileCardReadOnlyListDemo />
</ComponentPreview>
