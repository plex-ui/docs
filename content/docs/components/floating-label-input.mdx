### Clear button

<ComponentPreview details={<>Use the <code>onClear</code> prop to display a built-in clear button. The button appears when the input has a value and is not disabled or read-only.</>} code={`const [value, setValue] = useState("clearable@example.com")

<FloatingLabelInput
  label="Email address"
  value={value}
  onChange={(evt) => setValue(evt.target.value)}

/>`} previewMinHeight={168}>
