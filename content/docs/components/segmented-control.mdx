### Sizing & roundness

<ComponentPreview details={<>Sizing of the control is controlled with <code>size</code> and <code>gutterSize</code> props. <code>size</code> sets the control to a specific <code>height</code>, and <code>gutterSize</code> sets the horizontal padding of the inner options.<br/><br/>Each <code>size</code> provides default values for <code>gutterSize</code>, as well as <code>font-size</code> and <code>border-radius</code>. Each of these values can be overridden independently.<br/><br/>Create a fully rounded control with the <code>pill</code> prop. When <code>pill</code> is set, extra horizontal gutter is dynamically applied to the inner options.</>} code={`<SegmentedControl size="xl" pill={false} value={view} onChange={setView} aria-label="Select view">
  <SegmentedControl.Option value="all">All</SegmentedControl.Option>
  <SegmentedControl.Option value="failed">Failed</SegmentedControl.Option>
  <SegmentedControl.Option value="successful">Successful</SegmentedControl.Option>
</SegmentedControl>`} previewMinHeight={PREVIEW_MIN_HEIGHT.control}>
  <SegmentedControlSizingDemo />
</ComponentPreview>

### Scrollable

<ComponentPreview details={<><code>SegmentedControl</code> can support scrolling, but is not enabled by default.<br/><br/>To create a scrollable segmented control, wrap the component in a simple <code>flex</code> wrapper, which will then cause the control to create inner scrolling when applicable. When scrollable, the selected option will automatically be scrolled into view.</>} code={`<div className="max-w-[400px]"><div className="flex">...</div></div>`} previewMinHeight={PREVIEW_MIN_HEIGHT.control}>

### Narrow pill

<ComponentPreview details={<>When using the <code>pill</code> prop with narrow content (such as a single digit or an icon), especially when using large component sizes and small <code>gutterSize</code> values, the option width will automatically be at least equal to the control height.</>} code={`<SegmentedControl pill size="3xl" gutterSize="2xs">...</SegmentedControl>`} previewMinHeight={PREVIEW_MIN_HEIGHT.controlLarge}>
