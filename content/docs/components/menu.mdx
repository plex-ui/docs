### Checkbox items

<ComponentPreview details={<>Use <code>Menu.CheckboxItem</code> with <code>checked</code>, <code>onCheckedChange</code>, and <code>{"onSelect={(evt) => evt.preventDefault()}"}</code> to keep the menu open.</>} code={`const [settings, setSettings] = useState({
  showGrid: true,
  showLabels: false,
  enableShadows: false,
})

return (
  <Menu>
    <Menu.Trigger>
      <Button variant="ghost" color="primary">
        Checkbox menu <ChevronDownMd />
      </Button>
    </Menu.Trigger>
    <Menu.Content minWidth={200}>
      <Menu.CheckboxItem
        checked={settings.showGrid}
        onCheckedChange={(checked) => setSettings((s) => ({ ...s, showGrid: checked }))}
        onSelect={(evt) => evt.preventDefault()}
        indicatorPosition="end"
        indicatorVariant="solid"
      >
        Show grid lines
      </Menu.CheckboxItem>
      <Menu.CheckboxItem
        checked={settings.showLabels}
        onCheckedChange={(checked) => setSettings((s) => ({ ...s, showLabels: checked }))}
        onSelect={(evt) => evt.preventDefault()}
        indicatorPosition="end"
        indicatorVariant="solid"
      >
        Display labels
      </Menu.CheckboxItem>
      <Menu.CheckboxItem
        checked={settings.enableShadows}
        onCheckedChange={(checked) => setSettings((s) => ({ ...s, enableShadows: checked }))}
        onSelect={(evt) => evt.preventDefault()}
        indicatorPosition="end"
        indicatorVariant="solid"
      >
        Enable shadows
      </Menu.CheckboxItem>
    </Menu.Content>
  </Menu>
)`} previewMinHeight={168}>
  <MenuWithCheckboxesDemoWithControls />
</ComponentPreview>

## Reference

**Note:** Menu prevents `Tab` from advancing focus. For generic popover content use [Popover](/docs/components/popover) instead.

**Menu** (root)

<PropsTable data={[
  { prop: "forceOpen", type: "boolean", description: "Keep menu open (controlled)" },
  { prop: "onOpen", type: "() => void", description: "Called when menu opens" },
  { prop: "onClose", type: "() => void", description: "Called when menu closes" },
  { prop: "modal", type: "boolean", description: "Whether menu is modal (focus trap)" },
]} />

**Menu.Trigger** — The element that toggles the menu (e.g. a Button). No specific props.

**Menu.Content**

<PropsTable data={[
  { prop: "width", type: "number | string", description: "Panel width (px or CSS value)" },
  { prop: "minWidth", type: "number | string", default: "auto", description: "Minimum panel width" },
  { prop: "side", type: '"top" | "right" | "bottom" | "left"', description: "Preferred placement" },
  { prop: "align", type: '"start" | "center" | "end"', description: "Alignment relative to trigger" },
]} />

**Menu.Item** — List item. Use `onSelect` for actions; omit for non-interactive content.

<PropsTable data={[
  { prop: "onSelect", type: "(event: Event) => void", description: "Called when item is selected" },
  { prop: "disabled", type: "boolean", description: "Disable the item" },
]} />

**Menu.Link** — Item that navigates (uses your app link component). Pass `href` and link props.

**Menu.Separator** — Divider between items. No props.

**Menu.Sub** / **Menu.SubTrigger** / **Menu.SubContent** — Submenu wrapper, trigger, and content panel.

**Menu.CheckboxItem**

<PropsTable data={[
  { prop: "checked", type: "boolean", description: "Checked state" },
  { prop: "onCheckedChange", type: "(checked: boolean) => void", description: "When checkbox toggles" },
  { prop: "onSelect", type: "(event: Event) => void", description: "Use with preventDefault() to keep menu open" },
  { prop: "indicatorPosition", type: '"start" | "end"', description: "Checkbox position" },
  { prop: "indicatorVariant", type: "string", description: "Badge/indicator style (e.g. solid)" },
]} />

**Menu.RadioGroup** / **Menu.RadioItem** — Radio group: `value`, `onChange`; RadioItem: `value`, `disabled`.

**Menu.ItemActions** / **Menu.ItemAction** — ItemActions wraps hover actions; ItemAction accepts `onClick`.
