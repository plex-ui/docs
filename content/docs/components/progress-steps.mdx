## Reference

### `ProgressSteps`

<PropsTable data={[
  { prop: "current", type: "number", description: "Current active step (1-based)" },
  { prop: "orientation", type: '"horizontal" | "vertical"', default: '"horizontal"', description: "Layout direction" },
  { prop: "size", type: '"sm" | "md" | "lg"', default: '"md"', description: "Step indicator size" },
  { prop: "color", type: '"default" | "success"', default: '"default"', description: "Color scheme for completed/active steps" },
  { prop: "connectorStyle", type: '"solid" | "dashed"', default: '"solid"', description: "Connector line style" },
  { prop: "variant", type: '"minimal"', description: "Compact bar variant (requires total)" },
  { prop: "total", type: "number", description: "Total steps for minimal variant" },
]} />

### `ProgressSteps.Step`

<PropsTable data={[
  { prop: "icon", type: "ReactNode", description: "Custom icon instead of step number" },
]} />

### `ProgressSteps.Title`

Wrapper for step title text. Applies appropriate styling based on step state.

### `ProgressSteps.Description`

Wrapper for step description text. Applies appropriate styling based on step state.
