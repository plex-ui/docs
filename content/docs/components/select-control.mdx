### Country code selector

  );
};

<Select
  value={country}
  options={countries}
  onChange={(option) => setCountry(option.value)}

## Reference

Use `SelectControl` with any floating UI (e.g. `Menu`, `Popover`). The control renders the trigger; you provide the dropdown content.

<PropsTable data={[
  { prop: "selected", type: "boolean", description: "Whether a value is selected (affects placeholder styling)" },
  { prop: "variant", type: '"soft" | "outline" | "ghost"', default: '"outline"', description: "Visual variant" },
  { prop: "size", type: "string", description: "Control size" },
  { prop: "pill", type: "boolean", description: "Fully rounded trigger" },
  { prop: "StartIcon", type: "Component", description: "Icon at start" },
  { prop: "onClearClick", type: "() => void", description: "Enables clear button when set" },
  { prop: "dropdownIconType", type: '"chevronDown" | "dropdown" | "none"', description: "Dropdown indicator style" },
  { prop: "invalid", type: "boolean", description: "Invalid state" },
  { prop: "disabled", type: "boolean", description: "Disable control" },
  { prop: "block", type: "boolean", description: "Full width" },
  { prop: "loading", type: "boolean", description: "Loading state" },
  { prop: "opticallyAlign", type: '"start" | "end"', description: "Optical alignment" },
]} />

SelectControl shares the unified 9-step size scale with Button, Input, Select, and SegmentedControl.
