Plex UI ships with its own icon set used by every component — `Button`, `Input`, `Menu`, `Tabs` and so on default to these. Hover over any icon below to copy its import statement or raw SVG.

```tsx
import { ChevronDownMd, Search, Plus } from '@plexui/ui/components/Icon';

<Search />        {/* stroke variant */}
<ChevronDownMd />
<Plus />
```
