Plex UI ships with its own icon set used by every component — `Button`, `Input`, `Menu`, `Tabs` and so on default to these. Click any icon below to copy its import statement, raw SVG, JSX usage, or download the SVG file.

```tsx
import { ChevronDownMd, Search, Plus } from '@plexui/ui/components/Icon';

<Search />        {/* stroke variant */}
<ChevronDownMd />
<Plus />
```
